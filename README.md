# Interrogation Room

このリポジトリは、Next.jsを用いた尋問ゲームアプリ「Interrogation Room」のソースコードおよびドキュメントを管理しています。

## 概要

ユーザーが容疑者を尋問し、事件の真相を解明するWebアプリケーションです。
